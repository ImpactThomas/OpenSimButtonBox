Modular Open Sim Button Box - Documentation

The Modular Open Sim Button Box is a customizable buttons box based on a Teensy 3.2 running Arduino compatible code. The button box supports a basic layout with 17 buttons and 3 rotary encoders or an advanced layout with support for up to 32 buttons and 3 rotary encoders.
This project provides 3D printable models for the enclosure but you may also use the main board with any other box.
