  1. Soldering the PCB is shown in the following YouTube videos.
    Below is a list of the required parts
    • 4x 74HC4051 multiplexer DIP-16 (NOT 74HCT4051)
    • 4x 5V/100mA NPN (CBE) transistors TO-92
    • 2x PTC fuse 5mm leads Ihold:0.30/Itrip0.60A
    • 1x IC socket 28pins 15.24 mm (for Teensy)
    • 3x 10K resistor network 10pin(9+1) (2.54mm star configuration)
    • 1x 10K resistor network 6pin(5+1) (2.54mm star configuration)
    • 1x 100K 1/4 watt resistor
    • 1x 10K 1/4 watt resistor
    • 5x 10K 1/4 watt resistors OR 1x 10K resistor network 10pin 2.54mm parallel configuration)
    • 5x 1K 1/4 watt resistors OR 1x 1K resistor network 10pin 2.54mm parallel configuration)
    • 1x pin header straight 1x2 2.54mm
    • 1x pin header angled 1x2 2.54mm
    • 1x pin header angled 1x6 2.54mm
    • 2x pin header angled 2x3 2.54mm
    • 1x pin header angled 2x32 2.54mm
    • OPTIONAL: 1x pin header angled 1x5 2.54mm (for "Extra Pins")
    • 1x 5cm hookup wire
    • 1x pin header straight 1x1 female connector (to 5V on Teensy)
    • 1x LED with wire (user feedback LED)
    • 1x 10V 100uF radial capacitor
