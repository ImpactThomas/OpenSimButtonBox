Push Button Sources and Specifications
--------------------------------------



IMPORTANT
---------
All push buttons must be momentary push-button switches. The operation must be "push" is "ON", "release" is "OFF".
The common way to indicate a button like this is (ON)/OFF/(ON). The parentesis around "ON" indicate the momentary operation of the button.

A "ON/OFF/ON" button on the other hand would require a second push to turn "Off" again. The "ON/OFF"ON" type IS NOT compatible!


Specification
-------------
* 1x 16mm mounting hole for ignition switch
* 12-13mm mounting holes
* must support 4mm mounting plate
