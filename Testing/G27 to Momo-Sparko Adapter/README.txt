This adapter is for testing only! The G27 is not strong enough to handle 
a heavier wheel while still delivering proper force feed back.

I am using this adapter to temporarily mount my Beltenick steering wheel so I can develop my 
3D printed magnetic paddle shifters.


Dimensions are based on the following template
https://www.dropbox.com/s/hmzmvewwpmdi0c5/Logitech%20G25_G27%20Steering%20Wheel%20Hub%20Template.pdf#
